
# Keep track of it

A web application to control expenses over time.


## Installation/Dev

Launch web client

```bash
  npm install
  npm run dev
```
    
## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`VITE_API_KEY`: Firebase API KEY
